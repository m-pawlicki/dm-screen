# DM-Screen

A terminal application for creating character sheets from Dungeons & Dragons. Create, search, list, and delete characters from a locally stored sqlite database.

# Requirements

`uv` for venv management, which can be found [here](https://docs.astral.sh/uv/getting-started/installation/)

# Installation

Initialize the environment: `uv init`

Start the application: `uv run main.py`
